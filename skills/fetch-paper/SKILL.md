---
name: fetch-paper
description: |
  搜尋 arXiv 學術論文並以繁體中文摘要輸出為 Markdown。
  當使用者提到「搜尋論文」、「找論文」、「arXiv」、「學術搜尋」、
  「查詢研究」、「找最新研究」等情境時，主動觸發此 skill。
  即使使用者只說「幫我找 XXX 相關的論文」也應使用此 skill。
---

## 流程概覽

1. 執行 `fetch_paper.py` 抓取論文原文摘要
2. 逐篇閱讀摘要，以繁體中文撰寫精簡摘要與關鍵重點
3. 將結果整合輸出為完整 Markdown 檔案

---

## 步驟一：執行搜尋腳本

```bash
python skills/fetch-paper/fetch_paper.py "<關鍵字>" -n <數量>
```

參數說明：
- `<關鍵字>`：英文關鍵字效果最佳（例如 `"large language models"`）
- `-n`：論文數量，預設 5，上限 20

輸出檔案自動命名為 `output/YYYYMMDD_<關鍵字>.md`，例如 `output/20260224_LLM.md`。

---

## 步驟二：以繁體中文撰寫摘要

讀取腳本產生的 Markdown 檔案，對每篇論文補充繁體中文摘要，格式如下：

```markdown
### 繁體中文摘要

**精簡摘要**
（2–3 句，說明本論文的核心貢獻與方法）

**關鍵重點**
- （重點一）
- （重點二）
- （重點三）
```

摘要要精準傳達論文的研究問題、方法與貢獻，避免直譯原文、使用讀者易懂的繁體中文表達。

---

## 步驟三：輸出最終 Markdown

最終檔案結構範例：

```markdown
# arXiv 論文搜尋結果

**搜尋關鍵字**：`diffusion models`
**搜尋時間**：2024-01-01 12:00
**論文數量**：5 篇

---

## 1. 論文標題

- **作者**：Author A, Author B 等 5 位作者
- **發表日期**：2023-10-01
- **arXiv 連結**：[2310.00001](https://arxiv.org/abs/2310.00001)
- **PDF**：[下載](https://arxiv.org/pdf/2310.00001)

### 繁體中文摘要

**精簡摘要**
本論文提出...

**關鍵重點**
- 提出了...
- 在...任務上達到...
- 開源程式碼於...

### 原文摘要

> We present...

---
```

將最終結果寫回同一個 `.md` 檔案，並告知使用者檔案路徑。
